## 💡 What is BizAI?

BizAI is a fully working SaaS AI tool built for the **33 million small business owners** in the USA who waste hours every week on writing tasks.

Instead of spending hours writing emails, proposals, social media posts and customer replies — they just describe what they need and the AI writes it in seconds.

**No tech skills needed. Works on any device. Ready to use immediately.**

## 🚀 How to Run It

### 1. Clone the repo
```bash
git clone https://github.com/Sunnyhii/bizai-tool.git
```

### 2. Add your API keys
Open `bizai-app-full.html` and replace:
```javascript
const GEMINI_API_KEY = 'YOUR_GEMINI_API_KEY';
const STRIPE_PUBLISHABLE_KEY = 'YOUR_STRIPE_KEY';
```

### 3. Deploy
Upload to Vercel or any static hosting — live in 2 minutes.
